# Calc the Angle

This is a simple React + Vite + TypeScript app with React Router.

## Features
- Main page with a form to enter an angle
- Show page displays the entered angle

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
